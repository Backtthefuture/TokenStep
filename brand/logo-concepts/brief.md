# TokenStep Logo Brief

Product read: TokenStep is a macOS menu bar app that turns AI token usage into a daily progress ring, like step tracking for AI work.

Brand keywords: local, visible, calm, progress, daily goal, AI work, menu bar utility.

Generated directions:

1. Progress Orbit - closest to the current icon, cleaner and more brand-like.
2. Token Steps - token blocks as ascending steps inside a goal arc.
3. Local Lap - local/private core with a measured daily lap.
4. Usage Pulse - dashboard bars wrapped by the progress ring.
5. Infinite Lap - agent work looping past the daily goal.
6. Menu Compass - tiny-readability direction for macOS menu bar identity.
7. Hundred Million - daily goal token at the center with usage ticks around it.
8. Data Footprint - the step metaphor made from token blocks.

Suggested shortlist: 01 Progress Orbit for continuity, 02 Token Steps for metaphor clarity, 06 Menu Compass for macOS/product polish.
